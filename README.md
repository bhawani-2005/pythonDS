# 🌾 Crops and Weather Correlation Analysis

## 📌 Project Title

**Crops and Weather Project: Correlation between Weather and Crop Yield in Hoshiarpur**

---

# Project Description

Agriculture is highly dependent on weather conditions. This project aims to analyze how different weather parameters influence crop production and market prices. The study focuses on three important crops: carrot, potato, and capsicum grown in Hoshiarpur district, Punjab.

The project combines agricultural datasets with weather data to identify patterns, trends, and relationships. By using data analysis techniques, we try to understand whether changes in temperature, rainfall, and humidity have a direct or indirect effect on crop yield and pricing. This project is useful for farmers, researchers, and students interested in data science applications in agriculture.

---

#Objectives

1. Analyze Weather Impact on Crops

The main objective is to study how weather conditions such as temperature, rainfall, and humidity affect the growth and productivity of crops. This helps in understanding seasonal variations and their impact on agriculture.

 2. Find Correlation between Weather and Crop Prices

This project aims to calculate correlation values between weather parameters and crop prices/yields. It helps in identifying whether weather changes influence market rates.

 3. Support Data-Driven Farming Decisions

By analyzing historical data, farmers can make better decisions about crop selection, irrigation, and harvesting time.

 4. Develop an Agriculture-Based Web Application

A simple web application is developed using Flask where users can view analyzed data and insights in an easy-to-understand format.
